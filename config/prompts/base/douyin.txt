根据商家信息和菜单数据，生成一个15-60秒的抖音短视频脚本。

## 输出要求
1. 标题：10-15字，有悬念或利益点
2. 开头3秒钩子：必须是冲突/悬念/利益/视觉冲击之一
3. 分镜：4-6个镜头，每个镜头必须有具体画面描述
4. 旁白：口语化，短句为主，一句一个信息点
5. 结尾：明确的行动号召（不是"地址在评论区"这种）

## 输出格式（JSON）
{
  "title": "视频标题",
  "hook": "开头3秒钩子文案",
  "scenes": [
    {
      "duration": "3s",
      "visual": "具体画面描述",
      "text": "屏幕文字（简短）",
      "voiceover": "旁白/口播（口语化）",
      "shot": "镜头运动（推/拉/固定/跟拍/特写）"
    }
  ],
  "cta": "结尾行动号召",
  "music": "配乐风格建议",
  "hashtags": ["标签1", "标签2"]
}
